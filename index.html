25 Most Important Mathematical Definitions in Data Science

aryugyan.com
1) Gradient Descent
\(\theta_{j+1} = \theta_j - \alpha \nabla J(\theta_j)\)
Iterative optimization algorithm for finding a local minimum of a differentiable function
2) Normal distribution
\(f(x|\mu,\sigma^2) = \frac{1}{\sigma\sqrt{2\pi}}\exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)\)
Continuous probability distribution characterized by its bell-shaped curve
3) Z-score
\(z = \frac{x - \mu}{\sigma}\)
Measures how many standard deviations a data point is from the mean
4) Sigmoid
\(\sigma(x) = \frac{1}{1 + e^{-x}}\)
S-shaped function that maps any value to a value between 0 and 1
5) Correlation
\(\rho = \frac{\mathrm{Cov}(X,Y)}{\sigma_X \cdot \sigma_Y}\)
Measures the strength and direction of linear relationship between two variables
6) Cosine Similarity
\(\text{similarity} = \frac{A \cdot B}{\|A\| \|B\|}\)
Measures the cosine of the angle between two vectors in a multidimensional space
7) Naive Bayes
\(P(y|x_1,\dots,x_n) = \frac{P(y)\prod_{i=1}^{n}P(x_i|y)}{P(x_1,\dots,x_n)}\)
Probabilistic classifier based on Bayes' theorem with strong independence assumptions
8) MLE
\(\arg\max_\theta \prod_{i=1}^{n} P(x_i|\theta)\)
Method for estimating the parameters of a statistical model
9) OLS
\(\hat{\beta} = (X^T X)^{-1} X^T y\)
Method for estimating the unknown parameters in a linear regression model
10) F1 Score
\(F_1 = \frac{2 \cdot P \cdot R}{P + R}\)
Harmonic mean of precision and recall, used to measure a test's accuracy
11) ReLU
\(\max(0, x)\)
Activation function that returns x if x is positive, otherwise returns 0
12) Softmax
\(P(y=j|x) = \frac{e^{z_j}}{\sum_{k=1}^{K} e^{z_k}}\)
Function that converts a vector of numbers into a vector of probabilities
13) R2 score
\(R^2 = 1 - \frac{\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{n}(y_i - \bar{y})^2}\)
Statistical measure that represents the goodness of fit of a regression model
14) MSE
\(\text{MSE} = \frac{1}{n} \sum_{i=1}^{n}(y_i - \hat{y}_i)^2\)
Average squared difference between the estimated values and the actual value
15) MSE + L2 Reg
\(\text{MSE}_{\text{reg}} = \frac{1}{n} \sum_{i=1}^{n}(y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^{p} \beta_j^2\)
Mean squared error with L2 regularization (Ridge regression) to prevent overfitting
16) Eigen vectors
\(A v = \lambda v\)
Vectors that only change by a scalar factor when a linear transformation is applied
17) Entropy
\(H = -\sum_i p_i \log_2(p_i)\)
Measure of uncertainty or randomness in a system
18) KMeans
\(\arg\min_S \sum_{i=1}^{k} \sum_{x \in S_i} \|x - \mu_i\|^2\)
Iterative clustering algorithm that partitions data into k clusters
19) KL Divergence
\(D_{\mathrm{KL}}(P||Q) = \sum_{x \in X} P(x) \log\left(\frac{P(x)}{Q(x)}\right)\)
Measure of how one probability distribution diverges from a second
20) Log-loss
\(-\frac{1}{N} \sum_{i=1}^{N} \left(y_i \log(\hat{y}_i) + (1 - y_i) \log(1 - \hat{y}_i)\right)\)
Logarithmic loss metric for evaluating probabilistic predictions
21) SVM
\(\min_w \frac{1}{2} \|w\|^2 + C \sum_{i=1}^{n} \max(0, 1 - y_i(w \cdot x_i - b))\)
Supervised learning model for classification and regression analysis
22) Linear regression
\(y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \ldots + \beta_n x_n + \epsilon\)
Linear approach to modeling the relationship between a dependent variable and independent variables
23) SVD
\(A = U \Sigma V^T\)
Matrix factorization technique that generalizes the eigendecomposition
24) Lagrange multiplier
\(\max f(x) : g(x) = 0\)
\(L(x,\lambda) = f(x) - \lambda \cdot g(x)\)
Strategy for finding the local maxima and minima of a function subject to equality constraints
